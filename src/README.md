# ETF Alpha 投资分析平台

## 项目简介
ETF Alpha 是一个专为专业投资者与初级用户设计的现代化 ETF 数据分析与量化决策工具。它结合了传统技术指标、前沿的**多周期共振（Multi-period Resonance）**策略，以及 **Gemini AI 驱动的实时资讯与情绪分析**。通过实时数据追踪、智能形态识别及 D3.js 深度可视化，帮助用户在 A股、港股及跨境 ETF 市场中捕捉高胜率的交易机会。

## 核心功能模块与技术逻辑

### 1. 市场仪表盘 (Dashboard)
**功能定位**：全市场 ETF 的实时概览、筛选与监控中心。
- **全景监控**: 覆盖数百只主流 ETF（宽基、行业、跨境、红利等），实时计算各项技术指标。
- **数据获取逻辑**: 通过 JSONP 跨域请求新浪/腾讯财经 API，获取实时盘口数据与历史 K 线，并利用 IndexedDB 进行本地缓存以提升加载速度。
- **智能决策信号**:
  - **健康度 (Health)**: 基于对称性、趋势和波动率的综合评分。
  - **信号计数**: 实时统计买入/卖出因子数量。
  - **投资建议**: 明确输出 `OBSERVE` (观察)、`BUY_POINT` (建仓)、`HOLD` (持仓) 等状态。
- **高级筛选器**:
  - **突破筛选**: 快速定位今日或昨日突破近30日最高价的强势标的 (🚀)。
  - **黄金值筛选**: 快速定位回调至黄金分割支撑位 (0.618) 的标的。
  - **形态筛选**: 支持筛选“放量上涨”、“突破下降趋势线”、“均线纠缠”等形态。

### 2. 深度分析详情页 (Detail Analysis)
**功能定位**：单只 ETF 的“显微镜”级技术剖析与 AI 智能决策中心。

#### 🤖 AI 智能分析引擎 (最新功能)
- **新手操作指南 (Beginner Dashboard)**:
  - **一句话解读**: 将复杂技术面转化为通俗易懂的策略建议。
  - **技术评分**: 0-100 可视化进度条（红/蓝/绿区分超买、震荡、超卖区）。
  - **盈亏比估算**: 动态计算当前价格到最近阻力位（上望）与支撑位（下看）的比例，直观展示风险收益比。
  - **仓位建议**: 结合 30 日平均振幅（波动率）和当前趋势，给出具体仓位控制比例（如 30%-50%）及操作频率建议。
- **AI 量化交易计划 (专业版)**:
  - **投资评级**: 积极买入、持有待涨、中性匹配、减持观望。
  - **风险等级**: 结合波动率与趋势判定（低、中、高风险）。
  - **关键点位**: 自动计算目标价位、止盈位、止损位（基于动态斐波那契支撑/阻力网络及近期极值点）。
- **AI 资讯与情绪分析**:
  - **获取逻辑**: 接入 Gemini 3.1 Pro 模型，并调用 `googleSearch` 工具实时检索该 ETF 及相关板块的最新全网资讯。
  - **分析维度**: 提取核心事件、评估影响时长与逻辑、量化情绪得分（-10 到 +10）、提示关键风险，并输出 Markdown 格式的深度研报。

#### 📈 画线规则及标注点逻辑 (D3.js 核心特性)
1. **平行趋势通道 (Linear Regression Channel)**
   - **逻辑**: 基于最小二乘法计算选定周期（如60日）价格的线性回归中轴，并根据最大价格偏离度绘制上轨（压力）和下轨（支撑）。
2. **动态斐波那契 (Dynamic Fibonacci)**
   - **逻辑**: 自动识别当前视图内的最高点与最低点，判断主趋势方向，绘制 38.2% / 50% / 61.8% 回撤支撑线或反弹压力线。
3. **策略买卖点 (Strategy Markers)**
   - **R0 观察点**: 收盘价高于平行趋势线上轨。
   - **R1 建仓点**: 突破上轨后，次日缩量回调确认。
   - **R6/R8 止盈**: 跌破关键斐波那契位或盈利超 30% 强制止盈。
   - **R9/R10 止损**: 跌破波段低点或跳空低开熔断。
4. **成交量分布 (Volume Profile)**
   - **逻辑**: 统计价格区间的成交量堆积，识别 POC (筹码峰)。

### 3. 独立回测系统 (Backtest System)
**功能定位**：策略沙盒，用于验证“趋势跟随+斐波那契”策略的有效性。
- **参数配置**: 自定义初始资金、趋势周期、止盈止损阈值、加仓倍数等。
- **策略引擎**: 模拟 R1-R2 建仓，R3-R4 分批止盈，动态移动止损风控。
- **结果输出**: 资金曲线、最大回撤、胜率及逐笔交易明细。

## 技术栈与架构规则

- **前端框架**: Angular v21+ (严格遵循 Zoneless 架构，全面使用 Signals 进行状态管理，无 `zone.js` 依赖)。
- **可视化引擎**: D3.js v7 (直接操作 SVG DOM，实现高性能的 K 线图、指标线及交互式十字光标)。
- **AI 大模型接入**: `@google/genai` SDK (使用 `gemini-3.1-pro-preview` 模型，支持 Function Calling 与 Google Search Grounding)。
- **数据持久化**: 
  - `IndexedDB`: 用于海量历史 K 线数据的本地缓存，减少网络请求。
  - `LocalStorage`: 用于保存用户自选列表及 UI 偏好设置。
- **网络通信**: RxJS 结合 JSONP 规避跨域限制，内置自动重试与超时保护机制。
- **UI/UX 设计**: Tailwind CSS v4+ (响应式设计，移动端优先，支持暗黑模式扩展，大量使用语义化色彩如 `slate`, `emerald`, `indigo`)。
